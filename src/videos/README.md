# Video modules

Slide courses use a single runtime component. Scene courses regenerate per-module wrappers on activate.

**Scene course benchmark:** Module 1 of `agentic-ai-for-beginners` is the gold standard. See [`courses/BENCHMARK.md`](../courses/BENCHMARK.md).

```bash
npm run validate-benchmark -- agentic-ai-for-beginners 1
```

## Slide courses (current default)

| File | Role |
|------|------|
| `GenericModule.tsx` | Renders all slides from `moduleContent.ts` |
| `GenericModuleConfig.ts` | Duration/fps/width per module |
| `moduleContent.ts` | Generated by `activateCourse` from `courses/{id}/content.json` |
| `Root.tsx` | Registers `GenericModule` compositions |

Regenerate after content changes:

```bash
npx tsx scripts/activateCourse.ts <courseId>
# or
npx tsx scripts/generateModulesFromContent.ts all
```

`generateModulesFromContent` updates `Root.tsx` only (no `ModuleN.tsx` files).

## Scene courses (SVG + BaseDiagramScene)

Follow the **Module 1 benchmark** (`courses/BENCHMARK.md`): immersive premium SVGs, Gentle timings, `generateModulesFromScenes`.

```bash
npx tsx scripts/activateCourse.ts agentic-ai-for-beginners
npx tsx scripts/copySvgsToPublic.ts agentic-ai-for-beginners
npm run validate-benchmark -- agentic-ai-for-beginners 1
```

This runs `generateModulesFromScenes.ts`, which writes `ModuleN.tsx` into `src/videos/` and updates `Root.tsx` to use those components.

Timing constants: `scripts/sceneModuleTiming.ts` (0.5s scene buffer, 1.5s final tail, 0.57s whoosh).

## Archived generated modules

Previous `ModuleN.tsx` / `ModuleNConfig.ts` copies and timestamped backups live in `store/archived-videos/` and `store/archived-backups/videos/`.

## Timings

Word timings (Gentle) live in `courses/{id}/timings/` and sync to `public/timings/` on activate. Remotion loads them via `useModuleTimings`.
